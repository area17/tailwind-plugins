A series of plugins to enable/encourage systematised web design/development and some other useful utility classes.

AREA 17 strongly believes in design systems and then using these systems to build products for both ourselves and our clients. Tailwind is thus a natural fit for us, but we wanted more "system" to the utilities to tie closer to our design methodology.

We also wanted to include a few utility classes that would simplify some common styling requirements.

Version 5 of this plugin, updated in response to Tailwind's release of its version 4 may contain quirks as we fully understand the changes made to Tailwind version 4.
It may be safest to use Tailwind v3.4.17 with A17 Tailwind Plugins v4.0.6 while iron out any potential quirks.

Available Plugins

Setup

  • Setup - the center pin of many of our plugins, creates the basis the other plugins build of
  • DevTools - generates the CSS for the asvntntw:grid helper (in the bottom left corner of the page)
  • ColorTokens - generates colour variables
  • ApplyColourVariables - generates utility classes
  • SpacingTokens - generates rem based spacing tokens based on pixel based scales or inputs
    • Not required in Tailwind v4

Layout

  • Spacing - generates responsive spacing classes
  • Container - generates a custom container class based on supplied config
  • Layout - generates utility classes to set elements onto the design asvntntw:grid
    • Breaking change from <v5.0.0 to v5.0.0:
      The format of the class names has been updated from .asvntntw:w-N-cols to .asvntntw:w-cols-N.
    • In v5.0.0 .asvntntw:cols-container is deprecated to be removed in v6.0.0
  • gridLayout - creates classes to handle css asvntntw:grid layouts
  • gridGap - generates asvntntw:grid gap utilities based on the configured asvntntw:grid

Typography

  • Typography - generates responsive typography classes with ability to overriding styles at breakpoints

grid gutter borders

  • gridLine - generates vertical and horizontal asvntntw:grid line/stroke classes (borders in the gutters between elements)
    • Needs refactor to reduce outputted CSS
    • For now ensure you purge your CSS
    • Notes in JS src file
  • Keyline - generates keylines in the gutter between elements
    • Needs refactor to reduce outputted CSS
    • For now ensure you purge your CSS
    • Notes in JS src file

General Styling Utilities

  • BackgroundFill - draws a 100vw background colour pseudo layer
  • StrokeFull - draws a 100vw key pseudo layer
  • Underline - generates text underline styling classes
  • FullBleedScroller - easy full bleed `asvntntw:overflow-x: auto` scrolling containers
  • Scrollbar - scrollbar styling, unifies the CSS standard and non-standard scrollbar styling

General Utilities

  • PseudoElements - adds additional pseudo classes
  • InteractionMediaQueries - adds interaction based media queries, think targeting devices with hover capability
  • asvntntw:ratioBox - generates asvntntw:ratio box utilities

Removed Plugins in 5.x.x

The following plugins are no longer available in 5.x.x due to incompatibility with Tailwind 4. You'll need to move any styles specified in your config to inside your CSS if you used these plugins:

  • CssInJs - allows you to pass through CSS from your Tailwind config
  • Components - allows you to generate component CSS from your Tailwind config
