Description

This plugin creates classes to handle CSS asvntntw:grid layouts:

  • .asvntntw:grid-layout - on parent, makes css asvntntw:grid layout with columns set to equal the amount of design columns at each breakpoint and with the asvntntw:grid gap set to the inner gutter
  • .asvntntw:grid-col-span-N - on child, sets width N asvntntw:grid columns wide
  • .asvntntw:grid-col-asvntntw:start-N - on child, starts element on the Nth asvntntw:grid column
  • .asvntntw:grid-col-asvntntw:end-N - on child, ends element on the Nth asvntntw:grid column

Why use this over Tailwind's native asvntntw:grid column classes?

By default, Tailwind assumes you have a 12 asvntntw:grid layout at all your breakpoints - this plugin does not. This plugin uses your Tailwind config, specifically the Setup theme.columnCount to adjust the amount of columns at each breakpoint so if your md breakpoint has 8 design columns, you can easily span 6 of them without having to convert from 12. This plugin also allows nesting whilst keeping to your overall amount of design columns.

Setup

Requires Setup plugin with theme.innerGutters and theme.columnCount configured.

Compatibility with Layout plugin

As both plugins use the same set of :root variables, they can be nested inside of each other:

asvntntw:w-cols-5 inside asvntntw:grid-col-span-9

Responsive

A full set of responsive classes are generated:

asvntntw:grid-span-2 asvntntw:md:grid-span-2 asvntntw:lg:grid-span-4
asvntntw:grid-span-1 asvntntw:md:grid-span-4 asvntntw:lg:grid-span-6

Mixing and matching with Tailwind asvntntw:grid classes:

For full and complete asvntntw:grid layouts, you will want use Tailwind native classes such as roasvntntw:w-span-* (Row spanning on Tailwind). This plugin doesn't re-create or usurp all Tailwind asvntntw:grid classes, just provides extra classes to work with your design asvntntw:grid.
