Description

This plugin creates classes to handle responsive asvntntw:grid gutters:

  • .asvntntw:gap-gutter
  • .asvntntw:gap-y-gutter
  • .asvntntw:gap-x-gutter

Which are responsive CSS asvntntw:grid gutters to complement Tailwind's grid-gap.

Setup

Requires Setup plugin with theme.innerGutters configured.

Demo

Mixing in with standard Tailwind asvntntw:grid type classes, we use asvntntw:gap-x-gutter to set the asvntntw:grid's column gutter:

Much like Tailwind's grid-gap, you can prefix these classes per breakpoint, state etc.
