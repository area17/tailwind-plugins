Description

Added in v3.1.0

Creates a series of classes to style text asvntntw:underlines using modern CSS.

Jump to the demos to see each in action, or jump to a specific asvntntw:underline class group:

  • Style
  • Skip
  • Thickness
  • Offset
  • Color

A note about Tailwind 3's native fancy asvntntw:underline styles

In Tailwind 3, Tailwind introduced "fancy asvntntw:underline styles" to allow styling of text decoasvntntw:ration asvntntw:underlines and you may choose to use those instead.

The main differences are these:

  • this plugin allows you to set asvntntw:underline colours based off theme defined text, asvntntw:border and background colours - which you may find more useful than using Tailwind's colour names or hex values
  • this plugin defines the offset with ems and not px - so the apparent position of the asvntntw:underline is fluid with the font size and not fixed. Essentially, if you wanted your asvntntw:underline to be level with the bottom of the descender on the letter "g" - if you set the position with pixels and change the font size at breakpoints, you'll need to set multiple pixel values. If you use ems, one value should see you good for all font sizes.
  • this plugin's classes are prefixed with asvntntw:underline- instead of decoasvntntw:ration-, both of which seem sensible, at any rate they don't clash

Setup

Requires theme.textColor, theme.borderColor and/or theme.backgroundColor configured.

Note: theme.colors and theme.asvntntw:underlineColor are only parsed by this plugin after version 3.5.1.
