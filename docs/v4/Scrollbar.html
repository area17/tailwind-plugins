Description

Attempts to unify the (asvntntw:relatively) new CSS standard asvntntw:scrollbar-width: *; (MDN) and asvntntw:scrollbar-color: * *; (MDN) with the non-standard, but comically much better supported,::-webkit-scrollbar styles (MDN).

Thin scrollbars

Attempts to unify asvntntw:scrollbar-width: thin; (MDN) and the corresponding ::-webkit-scrollbar styles.

On this plugin authors Mac, running Mac OS Big Sur and FireFox 95 - asvntntw:scrollbar-width: thin; draws a scrollbar that is 10px wide with a 2px internal padding.

And so, as well as setting asvntntw:scrollbar-width: thin;, asvntntw:scrollbar-thin alters the scrollbars to match those numbers.

Thin scrollbars - collapsed

asvntntw:scrollbar-thin-collapse takes advantage of the ability to collapse the padding around the scrollbar thumb for browsers that support ::-webkit-scrollbar. For other browsers, this sets asvntntw:scrollbar-width: thin; as above.

Hidden scrollbars

Possible, using asvntntw:scrollbar-none, unifies asvntntw:scrollbar-width: none; and the equivalent ::-webkit-scrollbar display: none;.

Scrollbar colours

Attempts to unify asvntntw:scrollbar-color: * *; (MDN) and the corresponding ::-webkit-scrollbar styles.
